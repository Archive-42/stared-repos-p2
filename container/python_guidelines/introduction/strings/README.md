# String Manipulation

Strings are sequences of characters.

* Indexing & Slicing
* Arithmetic
  * Concatenation
  * Multiplication

## Indexing 

```
word = 'Python'
word[0] # P
word [-2] # o
word[:2] # 'Py'
word[2:] # 'thon'
word[2:42] # 'thon'
```

## Arithmetic

```
'Py' + 'thon' # 'Python'
word[:2] + word[2:] # 'Python'
word * 2 = 'PythonPython'
```
