What is the DOM
  Document Object Model - tree-like structure, representing our HTML


What is jQuery?
  - DOM manipulation JS dependency
  - a library of Java Script
  - one of the most used dependencies in JavaScript/HTML

FOUR uses of jQuery ($)
  1. selector - grabbing DOM elements 
      - css seelector (#id, .class, ul, li, header, etc.)
      - ex: $('li'), jQuery('.complete')
  2. creator - create DOM elements
      - ex: jQuery$('<li>')
  3. wrapper - wrapping DOM elements to use jQuery attributes
      - ex: $('li').last, $($('li')[0])
  4. setup doc ready callback
      - $(callback)
      - document.addEventListener('DOMContentLoader', callaback)

What is an event
