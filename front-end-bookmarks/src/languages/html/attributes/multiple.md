<h2 class="h3"><span>Code sample</span></h2>

```html
<label for="color">Select colors:</label>
<select id="color" multiple>
  <option value="red"> Red </option>
  <option value="green"> Green </option>
  <option value="blue"> Blue </option>
</select>
```
